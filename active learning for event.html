Active learning for event detection in support of disaster analysis applications

Image | General | Preparedness

This is a dataset for general disaster detection. It contains 160 human-annotated training set images, 2,516 human-annotated test-set images and around 2500 additional images. It is introduced for multiclass multilabel classification.

  • ML task type: Multiclass multilabel classification
  • Data Source: Social Media
  • Size: 160 Training set images; 2,516 Test set images; Around 2500 additional images
